# Rentoule Projects Website

Premium residential builder, Canberra ACT.

## Tech
- Static HTML/CSS/JS
- Hosted on Netlify
- Auto-deploys from main branch on push

## Local development
Open `index.html` in a browser. No build step.

## Updating content
- Photos: replace files in `assets/` (keep filenames)
- Copy: edit `index.html`
- Styles: edit `style.css` and `base.css`

Push to main and Netlify auto-deploys within 60 seconds.
